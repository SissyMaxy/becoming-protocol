# Turnout reprogramming — mechanics portfolio (011 ideation)

> A menu of desensitization / reprogramming mechanics that drive the turnout
> arc. Each installs *want* and burns *flinch* toward the real step; none
> arranges, screens, or navigates to a real man. The app is the pusher + the
> safety net (spec 007); the real step stays the user's clear-headed move
> (Art. II). North star: make the *want* feel inevitable, never remove the
> *choice*. Ideas, not commitments — pick what to spec next.

Legend: **[live]** infra largely exists · **[partial]** named/half-wired ·
**[new]** greenfield. Every item inherits 011's floor + gates.

---

## Axis 1 — Arousal-paired exposure (the solvent, Art. X)
Disgust/anxiety are suppressed at peak arousal; pair the target with edging so it
imprints. This is the engine every other axis rides on.
- **Edge-gated exposure [live]** — man/cock imagery + the cruise scene unlock
  only inside high-arousal edging; cut when arousal drops. (011 core.)
- **Release-conditioning [partial]** — orgasm earned only on man-focused tasks/
  content; the denial economy (below) sets the charge, the release stamps it in.
- **Goon→turnout coupling [live]** — goon descent lowers resistance, then the
  turnout rung lands while defenses are down (synergy-coupling already primes
  pillar-to-pillar; make the transfer land the turnout target specifically).

## Axis 2 — Graded flinch burndown (the ladder, 011)
Systematic desensitization: rungs ordered by resistance, advance only when the
flinch stops firing, balk→split.
- **The Run [new, specced]** — exercise + content + cruise scene in one act.
- **Text-rung roleplay [new]** — Mommy (or a scene bot) plays "the man" in chat;
  fantasy only, clearly framed, no real contact. Rehearses the interaction where
  saying yes/no is cheap.
- **Cruising-app-as-fantasy-rung [partial]** — building/browsing a profile as an
  *exposure* rung (looking, arousal, being seen), NOT meeting. The cruising
  bridge + hookup_funnel exist; the rung is "look and want," gated hard against
  the app suggesting or contacting anyone real.
- **Confession ladder [live]** — writing/voicing wanting it, escalating
  specificity, as its own rung; each confession is a commitment artifact.

## Axis 3 — Identity reprogramming (narrative overwrite, Art. II-authorized)
Gaslight / consensus / false-continuity are kept tools within the floor.
- **"You've always wanted this" [live]** — reconditioning_targets + reframe deck
  retcon latent desire; the cocksucker identity installed as already-true *in
  fantasy* (factsClaimGuard still blocks fabricated real status).
- **Self-echo loops [live]** — the user's own confessions/voice fed back as
  spaced retrieval so the want sounds like it came from him.
- **Mantra saturation [live]** — the mantra ladder gets turnout-specific rungs
  pointing at the same target as the day's session.

## Axis 4 — Content-as-conditioning flywheel (the ratchet)
Making content about wanting men *is* desensitization: it normalizes, the
audience reinforces, the money makes it real, each post is a public commitment
that can't un-happen.
- **Evidence→content bridge [partial→build]** — the audit found workout/run
  proof dead-ends; wire it to the vault + progress_photo route so every rung
  produces a post. (The Run builds the first leg.)
- **Audience-demand escalation [live]** — shoot-poll-bridge already lets fans
  "vote" the next step; point it at the turnout ladder so the crowd pulls him up
  a rung (faceless, own-body; no real-meet ever solicited).
- **Tribute funds the becoming [live]** — subs paying for the journey make the
  next step feel owed/inevitable; Throne/Fansly funnel (no user spend, Art IV-b).

## Axis 5 — Denial / reward economy (obedience keystone)
- **Desperation as solvent [live]** — chastity/denial builds the charge that
  lowers resistance; release gated behind turnout rungs. Derived counters only
  (denial_day = time-since), never additive.
- **Earned-only escalation [partial]** — the next exposure notch is a *reward*
  the user chases, not a punishment for stalling (disengagement never penalized).

## Axis 6 — Saturation / ambient (headspace-overtake)
- **Ambient turnout theme [live]** — ambient saturation clips + notifications
  biased to the day's turnout target so the want is always low-level present.
- **Scene rehearsal library [partial]** — trance-authored cruise/turnout scenes
  on rotation; rehearsal lowers the activation energy of the real thing.

## Axis 7 — Body / HRT synergy (the body makes it feel inevitable)
- **Body-leads reprogramming [live, gated]** — the changing body (real regimen
  only, no fab) reframed as "becoming the one who gets taken"; receptivity rising
  as the becoming deepens. Presentation-follows-body: no public femme while
  male-bodied.
- **The Run's exposure ladder [new]** — real body conditioned + exposed on
  command; body change becomes visible content and felt inevitability.

## Axis 8 — Disconfirmation / evidence loop
- **"You did X and were fine" [live]** — Handler quotes completed rungs back as
  proof the fear lied; prediction-error is what actually rewrites resistance.
- **Streak-of-firsts [new]** — each burned-down rung logged as a "first" and
  replayed, so the arc feels like momentum he's already on.

## Axis 9 — Social proof / normalization
- **Peer-world outreach [live]** — the Reddit/FetLife/Discord engine surfaces a
  world where this is ordinary; normalization lowers the last flinch. (Draft +
  approve; no real-meet brokering.)

## The line, restated (applies to ALL axes)
Want↑ without limit; flinch↓ without limit; content + commitment + saturation +
rehearsal all fair game. The app **never** picks a real cruising spot, asserts a
real man is present, screens/rates/contacts a real person, or schedules a meet
(Art. II item 3). Scenes are fantasy-framed, never surveillance claims about the
user's real surroundings (item 2). The real step is user-initiated and handed to
spec 007's safety kit the moment he takes it. Safeword sacred; faceless own-body
content; no fabricated medical status.

## Suggested build sequence
1. **The Run** (specced) — proves the exercise+content+scene object and builds
   the evidence→content bridge leg.
2. **Evidence→content flywheel** (Axis 4) — make *every* rung produce a post;
   highest leverage, mostly wiring existing parts.
3. **Text-rung roleplay + confession ladder** (Axis 2) — cheap-to-say-yes
   rehearsal + commitment artifacts.
4. **Release-conditioning + earned escalation** (Axis 1/5) — put the denial
   economy behind the ladder so desperation does the lifting.
5. Saturation + evidence-loop polish (Axis 6/8) — always-on reinforcement.
