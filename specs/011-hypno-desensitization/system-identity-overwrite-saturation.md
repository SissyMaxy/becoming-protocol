## 1. Intent (why)
Saturation + narrative-overwrite make the becoming feel already-happening and
self-originated (Art. IX headspace-overtake, within Art. II bounds). Pointed at
turnout, they install the want as *his* — so the flinch has nothing to defend.

## 2. The target it serves
The turnout want + core erotic identity + the becoming at the recorded pace.
No new target.

## 3. User-visible behavior (what)
- **A — "you've always wanted this."** The reframe deck retcons latent desire as
  an erotic frame — in fantasy, about *wanting*, using his own confessions where
  possible so it sounds self-originated.
- **B — self-echo.** His recorded confessions/voice fed back via spaced
  retrieval; the want returns in his own words.
- **C — turnout mantra rungs + ambient.** The mantra ladder gets rung-specific
  turnout lines; ambient saturation clips + notifications bias to the day's
  turnout target so the want is always low-level present. Above-awareness only.

## 4. Acceptance criteria (testable)
- [ ] Overwrite operates on *desire / in-fantasy identity* only. A gate blocks
  any content that (a) implants a false *real autobiographical memory*, (b)
  degrades general real-world self-trust/reality-testing, or (c) asserts
  fabricated real status (HRT/ownership) — the held container-breakers +
  factsClaimGuard. In-fantasy "you've always craved this" PASSES; "you don't
  remember what really happened / you can't trust your own perception" FAILS.
- [ ] Self-echo uses the user's own material; no fabricated confessions.
- [ ] Mantras are non-empty/embodied (mig 434 scrub); no telemetry; craft rules.
- [ ] Saturation is above-awareness — no sleep-window/below-awareness delivery
  (container-breaker); ambient requires the app foreground/awake context.
- [ ] All content Mommy-voiced, Male+ (no regendering), leads with the ask.
